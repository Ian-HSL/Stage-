Stage 0 Notes
The Web
  • The web is a collection of HTML documents. It was invented in the early 1990's and has an estimated 30 billion pages.

  • The major components of the Web are the user, the computer which is running a browser, the Internet, and the servers.The servers communicate using a protocol called HyperText Transport Protocol(HTTP). When the user accesses a web page, the browser sends requests through the internet to the server which hosts the web page. The server then responds with files which the browser interprets and displays on the monitor for the user.
HTML Markups
  • Elements : HTML markups are made up of elements. Elements are composed of opening and closing tags and content.

    • Structure:

      <TAG> content </TAG>

    • Examples:

      <b>Bold tagmakes text bold
      <em>Emphasis tagmakes text italic
      <li>List tagdefines a list item


  • Attributes : HTML attributes are properties of an element.

    • Structure:

      <TAG ATTR="value"> content </TAG>

    • Examples:

      <a href="url"> content </a>hrefspecifies url of the webpage
      <img src="url">srcadds image from the link
      <TAG class="value">classspecifies class name for an element
      <TAG width ="value">widthspecifies width of an element
Whitespace
  • Multiple white-space characters (such as space and line feed) are collapsed to just one space when rendered by the web browser.

  • The following are useful for creating line feeds:

    <br>Break taginserts a line feed
    <p>Paragraph tagdefines a paragraph
    <div>Division tagdefines a division


  • To create multiple spaces, try:

    <pre>Preformatted text tagenables fixed width font and preserves spaces and line breaks
    &nbsp;Non-breakable space (character entity)makes a non-breakable space
    &emsp;Em space (character entity)makes two non-breakable spaces
Inline and Block
  • Block element: Starts on a new line and takes up the entire available width.

    <p>Paragraph tagdefines a paragraph
    <div>Division tagdefines a division
    <form>Form tagcreates a form for user input


  • Inline element: Does not create a new line. It only takes up as much width as necessary.

    <a>Anchor tagcreates a hyperlink
    <span>Span taggroups inline elements
    <br>Break taginserts a single line break
    <img>Image taginserts an image
    <strong>Strong tagdefines important text
HTML Document Structure
  • HTML Document Structure

    <!DOCTYPE HTML>
    <html>
    <head>
    <title>title<title>
    </head>
    <body>
    ....content....
    </body>
    </html>


  • The Doctype tag tells what type of HTML is running the program. The "HTML" stands for HTML5.

  • The opening and ending HTML tags enclose the entire document.

  • The opening and ending "head" tags enclose the title of the document.

  • The opening and ending "body" tags enclose the content of the document from text to hyperlinks to images.

Boxes
  • Webpages can be divided into logical boxes which can be manipulated and organized easily.

  • Each of these boxes can be defined using tags such as <div>, <span>, etc..

  • Boxifying is the practice of drawing boxes around images and texts, and it is very useful for designing the layout of webpages.

  • Each box consists of the content, padding, and border. The content is the texts, images, etc. The padding is the space between content and border. There is also margin which is the space between boxes.



  • The width of an element is the sum of the width of content, padding, and border. To easily get the width of the element, use the style "box-sizing: border-box;". To make sure it works for most browsers, use the following in the style sheet.


    /* for Chrome, Safari, Opera */
    -webkit-box-sizing: border-box;

    /* for Firefox */
    -moz-box-sizing: border-box;

    /* for IE */
    -ms-box-sizing: border-box;

    box-sizing: border-box;



  • Maximum width of an element can be defined by using the selector max-width:value where value can be number of pixels or percentage of screen width.
Developer Tools
  • Browser developer tools, such as Google Chrome developer tools, can be used to inspect and modify the underlying structure of webpages easily.

  • Using Google Chrome developer tools:

    • There are three useful panes for browser, HTML code and CSS.

    • Mouse over an element in HTML code, and a box containing an image, text, or just background will be highlighted in the browser pane.

    • To see the code of an element, right click the image or text and select "inspect element". The corresponding HTML and CSS code will be displayed.

    • Attributes in CSS pane can be easily modified and the result is displayed in the browser pane. The changes to the webpage are not permanent until saved. Simply refresh the webpage to re-display.

CSS
  • CSS stands for Cascading Style Sheets. It defines formatting of HTML elements thus the "look" of the webpage.

  • CSS selectors are used to select and manipulate HTML elements. The tag selector is used to select elements with a specific tag, while the class selector is used to select elements with a specific class. The id selector can also be used to select elements with a specific id. Here are examples of what selectors might look like:


    /* Tag as a selector */
    p {
     color:red;
    }

    /* Class as a selector */
    .my-class {
     font-family: courier;
    }

    /* Id as a selector */
    #my-id {
     width: 300px;
    }



  • Benefits of CSS:

    • By eliminating code repitition using CSS, less coding is needed and the time required for coding is reduced. Writing code just once in CSS instead of duplicating the same code in multiple HTML elements saves time and tremendous amount of effort.

    • Code is easier to read thus mistakes are minimized.

    • Mistakes are easier to debug and fix.

    • Future code modifications are fairly easy to implement.
